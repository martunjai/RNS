README
=================

There are two parts of this demo app 

1. App bundle 
2. Data hosted in HTTP server


App bundle
---------
Main entry point - App.js 
    It has two pages - Login Page & VOD page 

Images - background image,poster image,thumbnail image,misc



Data hosted in HTTP server
-------------------------
data file - dataContent.json

1. goto path demo_apps/demo2
2. Run below command to launch HTTP server 
       python -m SimpleHTTPServer <port num>

       Ex : python -m SimpleHTTPServer 9081 

3. When App is launch, enter the http://<ip of machine>:<port num>  under "Server IP" text input box
